# Wichita City Council Documents Test Bed

This directory contains Wichita City Council agenda documents for testing and processing in the CIVIC-65 test bed branch.

## Contents

This directory includes PDF documents from Wichita City Council meetings dated April through October 2025.

## Purpose

This test bed branch (`Wichita-Notes-2025-test-bed`) is set up to:
- Test PDF processing and extraction workflows
- Validate document ingestion pipelines
- Process and store Wichita City Council documents in the database
- Develop and test document metadata extraction

## Files

All PDFs follow the naming convention: `Wichita_MM-DD-YYYY [Additional Description].pdf`
